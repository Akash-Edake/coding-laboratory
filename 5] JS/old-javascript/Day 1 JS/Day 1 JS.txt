	What is JS:
	It is client side scripting language which is mainly deal with events from user as well as graphical interface.
	It is an open source language. ==> Means u can contribute in that.
	
	How to run the JS:
	store file with .js extension.
	JS interprets the code in browser.

	Features:
		Lightweight 
		Interpreted ---> Line by line execution.
		netScape ---> LiveScript ---> JavaScript : ECMA Script: ES 1 ===> ES6/ES2015.
		Used to create static as well as dynamic web application.
		can insert HTML dyamically.
		Handle operaytions with events
		open source and cross platform.==> can work on any OS.
		Less server side interaction  ==> Maximum things will be handled at UI.
		Faster and immediate feedback.
		increase productivity or interactivity.
		Supported by all the browsers ==> chrome, firefox, IE, opera. Safari.
		JS gives engine to all browsers on which JS code will be executed. e.g for google chrome JS gives V8 engine.
		
	Limitations :
		can not perform multithreading ---> can not perform multiple operations at a time.
		can not used for network application.
		can not read or write files.
		Less security.
